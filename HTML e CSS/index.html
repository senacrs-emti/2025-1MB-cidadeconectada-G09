O que fazer

Verifique se o problema é geral, perguntando aos vizinhos ou observando outras casas.  Comunique à companhia responsável (de água ou energia) para relatar a falta e pedir informações.  Economize recursos: use a água ou energia restante apenas para o essencial.  Desligue aparelhos elétricos para evitar danos quando a luz voltar.  Use lanternas em vez de velas, evitando riscos de incêndio.  Mantenha geladeira e freezer fechados para conservar os alimentos.  Tenha sempre uma reserva de água potável e pilhas/lanternas para emergências futuras.
